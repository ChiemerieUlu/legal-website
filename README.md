# JusticeLegal Partners — Legal Website (Portfolio)

Simple static website built as a portfolio piece to showcase a legal firm's pages and layout.

**Project:** Basic legal website for portfolio use — includes a home page, about page, practice areas (services), and a small site footer/header with responsive nav.

**Tech:** HTML, CSS, JavaScript (vanilla)

**Contents:**
- index.html — main landing page
- about.html — firm info and attorney bios
- services.html — practice areas and FAQs
- assets/css/style.css — styles
- assets/js/main.js — small site JS (menu, FAQs, etc.)

**Preview locally:**
1. Open `index.html` in a browser or run a simple HTTP server in the repo root:

```sh
python3 -m http.server 8000
# then visit http://localhost:8000
```

**Notes:**
- Images included are from Pexels (placeholder images) and are hotlinked.
- Modify content, styles, and copy as needed for your portfolio.
